API NorthHost


Вы можете получать информацию а файлах с помощью API с форматом результата JSON.

Пример запроса получения информации о файле с ID 4444
GET http://{{ domain }}/api/infoFile/4444

Пример ответа на подобный запрос (JSON)
{"comment":"Test file", "download_link":"http://{{ domain }}/download?id=1", "downloaded":0, "filename":"file.txt", "id":4444,"is_private":false, "upload_date":"Tue,05 May 2020 21:32:16 GMT", "user_id":1, "username":"Admin"}
